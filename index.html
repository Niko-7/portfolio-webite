My Story So Far

  • Once upon a time

    When I was a small boy I can remember myself following my grandma around in the kitchen trying to understand what she was doing. As soon as I hit 18 I spent two years in culinary school and started working in different restaurants . After years of burns,scrapes and cuts I realised that I only loved cooking as a hobby.

  • How It All Started

    I was always a tech savvy and whenever I wasn't cooking I would spend my free time on my computer and so through a friend of mine I got introduced to coding, I can say it was love in the first sight! I Would spend hours online learning to code , it's amazing how you can spend hours working on a project and it only feels like a few minutes passed, coding has the power to make you feel like a kid rediscovering life!

  • From Chef To Dev

    Soon after i started coding i decided to take a leap of faith, say goodbye to my old life and joined the software development course at the Northcoders bootcamp. It can be very challenging at times but there is nothing more exciting and rewarding than solving a coding problem! The possibilities for learning an improving are endless and im glad I joined Northcoders because through their intense coding bootcamp im one step closer to achieving my goals.

  • Now

    With only a few weeks left on the course, I am more excited than ever as I will get to use my coding skills in a professional setting! My long-term planning can help build and advance growth initiatives within your organisation. I am committed in creating opportunities that can benefit my career and my future employer.

Built with HTML, CSS and JavaScript my portfolio is a work in progress. It's made with responsive design in mind since about 50% of internet traffic is mobile and tablet users. I used Vanta.js for the animated website background that only takes a few lines of code.

Pokemon Battle

A Pokemon battle game that you can play in your terminal! This game was built by using JavaScript(Object Oriented Programming) and the Inquirer library, this game can be played using inputs and selections to play out a Pokemon battle between 2 trainers.

Sedit

Back-End project of an API containing articles, topics, comments and users data. Created with postgreSQL, Knex and express server. Have been using Jest for testing throughout the project. This project is a work in progress, i will use React to build the user interface and interactive elements on the website. This project is currently functional as an API and being hosted with Heroku : https://mitch-mitch.herokuapp.com/
